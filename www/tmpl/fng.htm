HTTPS Fingerprint (SHA1)


On the ESP8266 platform this fingerprint is used for https communication with IAS.

This was introduced with library version 1.0.6
From 2.1.0 on we also support root certificates and BearSSL certificate bundle's which will become the future default.

By changing the HTTPS_8266_TYPE define in the library config.h file you can use root certificates instead of the current fingerprint.
